Introduction

Basic Philosophy:

We humans tend to learn the classification rules just from a few examples. All we need is someone to show examples of two to three images of a new breed dog and with these, we can recognize images of the same kind seamlessly. i.e. we just need few-shot training -- The ability to recognize images from a few set of training images.

How did we accomplish this?

We have a kind of good inductive bias or good priors that we have accumulated through a lot of experience. Can we achieve the same using machines as well?

Meta learning tries to answer the following question in some sense.

Can we explicitly learn priors from previous experience that lead to efficient downstream learning?

i.e. can we learn to learn?

Meta Learning set up

we don't have a set of data points but we have a set of training datasets.

Views of Meta Learning

There are two views of meta-learning

  1. Implementation view

    • Consume a dataset -- and fine tune
    • Make predictions on the new data points

  2. Understanding view

    • Use training datasets to learn a good prior
    • This prior enables the model to do few shot training easily.

Meta Learning Formal Definition

In supervised learning we do: argmaxϕlogP(ϕ|D)=argmaxϕlogP(D|ϕ)+logP(ϕ) (a regularizer)​

We ignore the denominator above as it is a normalizing constant and is not affect the argument of optimization

In meta learning, the set of training datasets is, Dmeta=(D1,D2,⋯,Dn) where Di={(xi,yi)}​

And the target training dataset is D={(xi,yi)}​

And the meta learning problem is: argmaxϕlogP(ϕ|D,Dmeta)​

i.e., find the parameters ϕ that produce the set of datasets {Dmeta∪D}​

Note, by assumption, D is very scarce and any attempt to perform argmaxϕlogP(ϕ|D)​ will likely overfit to D and fail during inference

Assumption - Let us assume that θ​ forms a sufficient statistic for ϕ w.r.t. Dmeta

i.e. ϕ⊥⊥Dmeta|θ

logP(ϕ|D,Dmeta)=log∫ΘP(ϕ,θ|D,Dmeta)dθ=log∫ΘP(ϕ|D,θ)∗logP(θ|Dmeta)dθ

Thus we can learn θ from Dmeta​ first and then use it as a good inductive bias to learn ϕ for D

To put this formally,

Let

(1)θ∗=argmaxθlogP(θ|Dmeta)

The meta learning problem is to perform argmaxϕlogP(ϕ|D,θ∗)​

Note: We actually had to maximize log∫ΘP(ϕ|D,θ)∗logP(θ|Dmeta)dθ=logEθ|Dmeta[P(ϕ|D,θ)]​

Because the above expectation is difficult to evaluate, one can do Monte-carlo sampling to approximate the estimate and then maximize it. But here, we kind of approximate it using one sample that is obtained as an argmax in the Equation (1)​

 

​​​TLDR; Meta learning procedure contains two steps:

  1. Meta Learning step - Compute θ∗=argmaxθlogP(θ|Dmeta)​
  2. Domain Adaptation - compute ϕ∗=argmaxϕlog(ϕ|D,θ∗)​
